# Vim Novak Theme!

 A dark colorscheme with colorful tokens!
 
## Screenshot
![novak theme](screenshot.jpg)
The font used in screenshot is [_JetBrains Mono_](https://github.com/JetBrains/JetBrainsMono)

## Installation With [vim-plug](https://github.com/junegunn/vim-plug)

Add this line to your vim/neovim config file:
```
Plug 'kouroshalinaghi/vim-novak-theme'
```
and then in vim, run:
```
:PlugInstall`
```
Run `:colorscheme novak` in normal mode.

If you want to make novak your default colorscheme add this line to your vim config file:
```
colorscheme parkour
```
